Linux常用指令（重制版）

Posted on 2020-07-18 | In Linux学习笔记 |

系统使用Ubuntu18.04，无图形界面，本地系统为Windows10

一、Linux系统指令

  • 进入指定文件夹
    cd /root/mygit/project1
  • 进入上层文件夹
    cd ..
  • 进入根目录文件夹(root)
    cd ~
  • 显示当前文件夹路径
    pwd
  • 显示当前文件夹里的文件和文件夹
    ls -l
  • 在当前文件夹下创建文件夹
    mkdir project2
  • 在当前文件夹下创建文件
    touch readme.txt
    touch gitCommand.md
    touch go.sh
  • 删除文件或文件夹
    sudo rm -r readme.txt
    sudo rm -r project2
  • 重命名文件或文件夹
    mv readme.txt README.txt
  • 将上述指令结合起来使用，你就可以非常方便的解决一些文件操作的小问题了。比如进入某个文件夹，新建一个子文件夹用来存放自己的markdown笔记，就可以在命令窗口里输入如下命令：
    cd /usr/note && mkdir LinuxNotes && touch day1.md

二、常用软件指令

  • screen
    • 新建一个新窗口
      screen -S name
      注意S是大写
    • 将当前窗口挂起（后台运行）
      ctrl + A + D
      注意这是快捷键，按后面的键时不要松开前面的键
    • 显示当前已经开启的窗口
      screen -ls
      显示的时候会显示窗口ID和名字，以及是否挂起
    • 进入挂起窗口
      screen -r name
      注意r是小写
    • 关闭窗口
      kill ID
      注意ID可以用screen -ls指令查看
  • zip
    • 解压文件到当前目录
      unzip file.zip
    • 压缩文件
      zip -r file_name.zip file.txt
      注意是将file.txt文件压缩为file_name.zip文件，不要弄反了

三、上传下载文件

  • rz/sz
    • 从服务器下载文件
      sz file.zip
    • 从本地上传文件
      rz
      注意本地需要相应软件，比如Xshell
  • windows命令窗口上传文件
    • pscp C:/fold1/fold1_2/file.txt root@xx.xxx.xxx.xxx:/usr/folderA

四、anaconda使用指令

  • 查看虚拟环境列表
    conda info -e
  • 创建虚拟环境
    conda create -n env_name
  • 进入虚拟环境
    conda activate env_name
  • 退出虚拟环境
    conda deactivate
  • 在当前虚拟环境下安装 包
    conda install appname
    conda install URL
